# react-admin-mall

该项目是利用网上已有的RESTful API，作为后端，然后使用react构建后台管理系统

## 技术选型

1. 前后端完全分离，分层架构，模块化开发
2. 单页应用，框架开发
3. React, React-Router, ES6, Sass
4. Webpack, Webpack-dev-server, Git
